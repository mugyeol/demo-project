# Todo Joseon tone & manner - 2026-03-04

## Context & Goals
The current interface screams neon sci-fi. The request is to reframe it with a Joseon Dynasty sensibility: warm hanji paper, dancheong pigments, wooden hardware, and brush-script typography. Functional scope stays the same (local storage todo list, filters, stats, clear completed). Success equals instantly recognizable Korean heritage vibes without sacrificing clarity on modern displays. Constraints derived from repo conventions: keep the React tree intact, rely on CSS/asset changes only, stay inside Vite + Google Fonts (no heavy font downloads), retain WCAG AA contrast for body text, and respect existing focus states. Tone references include royal court scrolls (silk gradients, gold foiling), palace eaves (teal/indigo/red patterns), and official records (stamp-like buttons). The experience should feel handcrafted yet crisp on retina screens. We must also consider `prefers-reduced-motion` (animations minimal) and mobile responsiveness.

## Direction Options
**Option A - Hanji Scroll.** Off-white parchment background with gentle fiber noise, ink-inspired typography (Song Myung headings, Gowun Batang body), and rounded rectangles mimicking scroll panels. Embossed indigo borders plus cinnabar seals for buttons. Pros: elegant, achievable via gradients/pseudo elements. Cons: risk of looking too plain without subtle textures. **Option B - Palace Court Ledger.** Deep navy background, gold typography, grid reminiscent of ledger tables; uses faux lacquer wood frames. Pros: dramatic, high contrast. Cons: dark mode may fight readability for long lists and requires more box shadows. **Option C - Folk Festival.** Bright dancheong motif, lots of SVG ornaments. Pros: playful; cons: busy, requires additional markup/assets. Recommendation: Option A best balances authenticity and maintainability while leaving room for micro-patterns inspired by palace screens.

## Experience Design (Visual Language & Layout)
Typography imports `Song Myung` 700 for headings/CTAs and `Gowun Batang` 400/600 for body, with `Noto Sans KR` fallback for UI labels. Root tokens shift to parchment neutrals (`--page-bg: #f7f1e1`, `--surface: rgba(255, 250, 240, 0.95)`), earthy inks (`--ink: #2c1a10`, `--muted: #6f5540`), dancheong pigments (`--accent: #b13a35`, `--teal: #0f5b4f`, `--indigo: #1e3a5f`), and brass highlights. `.todo-page` renders layered radial gradients plus a pseudo-element with `background-image: url(data:image/svg+xml,...)` to fake hanji fibers (low-opacity repeating-linear-gradient). `.todo-shell` gains subtle bevel, double border, and top/bottom scroll rods using `::before`/`::after` brown bars with drop shadow blur. Focus outlines become `2px solid var(--teal)` with 4px offset to match ink strokes. The header adopts vertical rhythm like a calligraphy title plaque: eyebrow becomes a jade ribbon with gold dividers, `h1` uses letter spacing reminiscent of seal script, and subtitle sits inside a small ink stamp background (#e5b8a0). Motion: only `fadeSlideUp` for list items (<300ms) and soft parallax shimmer on the scroll background; reduced-motion removes them entirely.

## Experience Design (Components & Interactions)
Form input style: cream base, thin charcoal stroke, slight inner shadow to mimic brush indentation; placeholder text italicized to read like penciled prompts. `Add` button transforms into a cinnabar seal with square shape, `text-transform: none`, and `::after` cross-hatching; hover lifts + uses multiply blending for depth. Filter pills become paper tags tied with string (rounded rectangles with notches via `conic-gradient` masks). `Clear done` button reinterpreted as transparent link with underline growing on hover to echo brush stroke. Todo items display alternating parchment shades, vertical calligraphy guidelines (very low-opacity lines), and a side ornament (small teal square). Checkbox styling: circular wax seals; unchecked shows embossed border, checked reveals golden check mark and stamping effect via `box-shadow`. Delete button iconography shifts to `X` script inside a bordered tab, with hover turning deep indigo. Empty state reads \"비어 있음 - 새로운 기록을 남기세요\" in italic; stats footer uses little pill badges resembling wooden tablets with engraved text. All text remains >= 15px with contrast > 4.5:1 on surfaces. Media queries reduce ornament sizes under 560px while keeping padding comfortable.
