# Todo Joseon style tone & manner — 2026-03-04

## Context & Goals
The existing neon synthwave aesthetic clashes with the request for a Joseon Dynasty-inspired todo interface. Users should feel as if they are managing tasks on a refined scholar's desk: tactile hanji textures, muted ink colors, careful brush calligraphy, and ceremonial flourishes. The information hierarchy (input, toolbar, list, footer) stays the same; only the visual treatment shifts. Constraints: keep the React markup intact, ship a single CSS file (plus Google Fonts if needed), remain responsive down to 360px, and maintain WCAG AA contrast on text in buttons and todo rows. The redesign should celebrate Joseon motifs—natural pigments, silk gradients, wooden framing—without resorting to kitsch or heavy imagery. Success criteria: (1) add layered textures and scrollwork that evoke hanji and lacquerware, (2) pair a display hangul serif with a refined supporting serif for readability, (3) convert controls into rectangular forms reminiscent of scroll labels and seal stamps, and (4) maintain subtle motion that mimics the slow reveal of sliding doors or ink unfurling.

## Direction Options
**Option A – Hanji Scholar Desk:** Creamy paper background (#fdf7ec), charcoal ink (#1e1b16), moss/sage accents (#6b7a4f), cinnabar red stamp (#c24132) for destructive actions. Introduce layered paper textures, embossed edges, and brushed dividers. Pros: historically grounded, tactile, minimal dependencies. Cons: risk of looking flat without enough layering.
**Option B – Royal Court Lacquer:** Deep indigo background (#0d1423) with gilded outlines (#c5a46d), jade green (#3b705c) highlights, and luminous lantern glows. Use darker theme with reflective surfaces and scroll rods. Pros: dramatic, luxurious. Cons: may impact readability if not balanced, heavier shadows.
**Option C – Folk Textile Patchwork:** Muted pastel squares, embroidered borders, and warm gradients referencing bojagi textiles. Pros: vibrant storytelling. Cons: can become visually noisy, harming focus on tasks. For clarity and accessibility, Option A is preferred; it echoes scholarly orderliness and easily translates into CSS gradients and pseudo-elements.

## Final Experience Design
Overall layout stays centered (max-width 720px) but appears as layered parchment resting on a wooden desk. Body background: radial gradient (#f1e4cf → #d8c3a5) with a `::before` noise texture and faint ink splatters. Typography: import `Song Myung` for headings and `Noto Serif KR` for body text (add `serif` fallbacks). Define CSS variables: `--paper`, `--paper-dark`, `--ink`, `--ink-muted`, `--accent-sage`, `--accent-stamp`, `--accent-gold`, `--border`, `--shadow-parchment`. Header: eyebrow replaced with vertical separator + small crest (SVG via pseudo) and letterspaced hanja-like text; H1 uses `Song Myung` 48px with `text-stroke` shadow to mimic brush edges, subtitle becomes bracketed romanization with dotted underline. Todo form: input uses paper texture, 2px dark border, subtle inner shadow; focus state reveals sage gradient line as if ink spreads. Primary button resembles scroll tag: rectangular with tied-knot pseudo elements and gradient from ivory to pale jade. Toolbar becomes wood inlay bar with segmented filter chips styled like narrow tablets; active state uses stamp red border plus inset shadow. Clear button styled as red seal block, uppercase old copperplate letter spacing.

Todos: each item is a deckled card with vertical ragged edge pseudo-element on the left. Checkbox replaced by circular ink stamp (brown). When checked, entire card gets sage wash with brush animation (keyframe to fade in). Delete button uses cinnabar fill with carved text effect. Empty state shows dashed brush circle with small haiku text. Footer stats rendered as hanging tags with thin cords (`::before` lines). Motion: fade+scale for parchment stacking, filter transitions use `clip-path` wipe, checkbox uses `scale` ripple. Keep reduced motion respect. Ensure focus states use 3px gold outline for accessibility.
